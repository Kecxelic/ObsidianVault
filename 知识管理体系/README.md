# 知识管理体系 

## 文件夹结构

```
AI知识库/
├── 技术类/
├── 工作类/
├── 生活类/
└── 想法类/
```

## 使用方法

### 文件命名规则
- 格式：`YYYY-MM-DD-主题.md`
- 例如：`2025-01-15-Python编程技巧.md`

### 内容模板（超简版）
```markdown
# 主题

## 核心要点
- 
- 

## 可行动项
- 
```

### 分类说明
- **技术类**：编程、工具、软件等
- **工作类**：项目、方法、流程等
- **生活类**：健康、效率、习惯等
- **想法类**：思考、灵感、创意等
